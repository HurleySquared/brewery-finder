# Brews 'R Us

## Description
Brews 'R Us is a Front End application that allows the user to locate Brewery's near them.

## Technologies Used
- Open Brewery DB
- Weather API (TBD)
- React.js
- Node.js
- JavaScript, JSX
- CSS3
