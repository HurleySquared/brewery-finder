# Brews 'R Us

## Description
Brews 'R Us is a full stack application that allows the user to locate Brewery's near them. With options to filter through, they can better choose which brewery fits their needs.

## Technologies Used
- Open Brewery DB
- Weather API (TBD)
- React.js
- Node.js
- JavaScript, JSX
- Database(TBD)
- Express
- CSS3
